CoinGecko

API

Watchers.CoinGecko.coinschart — Method

Get close price and volume for symbol id for specified number of days.

days="max"

returns all available history.

source
Watchers.CoinGecko.coinschart_range — Method

Same as CoinGecko.coinschart but for specified timerange.

  • from, to: date range boundaries.
From coingecko:

Data granularity is automatic (cannot be adjusted)

  • 1 day from current time = 5 minute interval data
  • 1 - 90 days from current time = hourly data
  • above 90 days from current time = daily data (00:00 UTC)
source
Watchers.CoinGecko.coinschart_tf — Method

Pull market data range according to timeframe.

  • 5m: 1 day
  • 1h: 90 days
  • >1h: 365 days
source
Watchers.CoinGecko.coinshistory — Function

Get data for symbol id at specified date.

source
Watchers.CoinGecko.coinsid — Method

Get all current data for symbol id.

source
Watchers.CoinGecko.coinsmarkets — Method

Get markets for a list of symbols, accepting params CoinGecko.Params.

source
Watchers.CoinGecko.coinsohlc — Function

Pulls ohlc data (no volumes) according to timeframe.

  • <=30m: 1 day
  • <=4h: 30 days
  • >4h: max
source
Watchers.CoinGecko.coinsticker — Method

Get all tickers (from all coingecko exchanges) for symbol id.

source
Watchers.CoinGecko.derivatives — Method

Returns all unexpired derivative contracts.

Contracts are grouped by exchange, the id being the slugified exchange name.

source
Watchers.CoinGecko.derivatives_from — Method

Fetch derivatives from specified exchange.

Returns a Dict{Derivative, Dict}.

source
Watchers.CoinGecko.globaldata — Method

Returns global data for:

  • volume: total_volume (Dict{String, Float64})
  • mcap_change_24h: marketcapchangepercentage24h_usd
source
Watchers.CoinGecko.idbysym — Method

Get all coingecko item id matching by its symbol.

source
Watchers.CoinGecko.idbysym — Method

Get the first coingecko item id by its symbol.

source
Watchers.CoinGecko.loadcoins! — Method

Load all coins symbols.

source
Watchers.CoinGecko.loadderivatives! — Method

Returns the list of all exchange ids.

source
Watchers.CoinGecko.ratelimit — Method

Allows only 1 query every 3000 milliseconds seconds.

source
Watchers.CoinGecko.tickers_from — Method

Fetches top 100 tickers from exchange.

Returns a Dict{Asset, Dict}
